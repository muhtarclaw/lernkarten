# Wortschatz — German A1·A2·B1 Trainer

A beautiful, interactive German vocabulary flashcard and quiz app. Practice German words across A1, A2, and B1 levels with multiple themes.

## Features

- **Flashcard Mode**: Flip cards to learn German-English translations
- **Quiz Mode**: Test your knowledge with multiple-choice questions
- **Progress Tracking**: Track streaks, XP, and mastery
- **Multiple Themes**: Browse vocabulary by category
- **Level Filters**: Focus on A1, A2, or B1 words
- **German Audio**: Auto-speak German words using Web Speech API
- **Beautiful UI**: Clean, modern design with smooth animations

## Technologies

- Pure HTML, CSS, and JavaScript (no frameworks)
- Web Speech API for German pronunciation
- Responsive design for mobile and desktop
